# 🐊 CroCode Marketplace

[![Website Live](https://img.shields.io/badge/Website-Live-brightgreen)](#)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**CroCode** adalah platform digital modern yang dirancang untuk menjual *source code* website dan aplikasi berkualitas tinggi. Website ini berfungsi sebagai katalog produk sekaligus gerbang utama transaksi bagi pembeli.

---

## 🚀 Fitur Utama Website

* ✨ **Modern Landing Page:** Menggunakan desain responsif berbasis **Bootstrap 5**.
* 🛍️ **Product Catalog:** Tampilan kartu produk yang bersih dengan label harga dan kategori teknologi (PHP, CodeIgniter, dll).
* 📱 **Mobile Friendly:** Optimal diakses melalui smartphone, tablet, maupun desktop.
* 💬 **Direct WhatsApp Integration:** Sistem pemesanan langsung yang terintegrasi dengan WhatsApp untuk konversi penjualan yang cepat.
* ⚡ **Smooth Animations:** Menggunakan pustaka **AOS (Animate On Scroll)** untuk pengalaman pengguna yang interaktif.

---

## 🛠️ Teknologi yang Digunakan

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Framework UI:** Bootstrap 5
* **Library Animasi:** AOS (Animate On Scroll), GLightbox, Isotope.js
* **Icons:** Bootstrap Icons

## 📝 Cara Menggunakan (Untuk Pengembang)

Jika Anda ingin melakukan kustomisasi atau menambah produk, ikuti langkah berikut:

1. **Clone repositori ini:**
```bash
git clone [https://github.com/rohmansyah23/Crocode.git](https://github.com/rohmansyah23/Crocode.git)
cd Crocode

```


2. **Tambahkan aset gambar:**
Masukkan gambar produk baru Anda ke dalam folder `assets/images/`.
3. **Tambah produk di HTML:**
Buka file `index.html` dan salin (duplikasi) blok kode berikut pada bagian `#sourcecode` untuk menambah item baru:
```html
<div class="col-lg-6 col-md-6 d-flex">
  </div>

```


4. **Update ke GitHub:**
Simpan perubahan Anda lalu unggah kembali dengan perintah:
```cmd
git add .
git commit -m "Tambah produk baru"
git push origin main

```
